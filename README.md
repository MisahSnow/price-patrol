# Price Patrol 📉

A sleek Chrome extension that tracks Amazon product prices and alerts you when they drop. Built with affiliate monetization in mind — every link uses your Amazon Associates tag so you earn commission on purchases.

![Price Patrol](icons/icon128.png)

## Features

- **One-Click Tracking** — Visit any Amazon product page and click "Track Price" to start monitoring
- **Real-Time Price Monitoring** — Automatically checks prices at configurable intervals
- **Desktop Notifications** — Get instant alerts when tracked products drop in price
- **Affiliate Integration** — All product links include your Amazon Associates tag for commission earnings
- **Price History** — Track highest, lowest, and current prices for every product
- **Beautiful Dark UI** — Modern, polished interface that's a joy to use
- **Multi-Region Support** — Works across all major Amazon domains (.com, .co.uk, .de, .fr, .es, .it, .co.jp, .in, .com.au, .com.br, .com.mx, .ca)
- **Smart Drops View** — Quickly see which products have the biggest price drops

## Installation

### From Source (Developer Mode)

1. Clone this repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/price-patrol.git
   ```

2. Open Chrome and navigate to `chrome://extensions/`

3. Enable **Developer mode** (toggle in the top right)

4. Click **Load unpacked** and select the `price-patrol` folder

5. The Price Patrol icon will appear in your extensions bar!

## Setup

1. Click the Price Patrol icon in your Chrome toolbar
2. Click the **Settings** gear icon
3. Enter your **Amazon Associates Tag** (e.g., `yourtag-20`)
4. Configure your preferred check interval and notification settings
5. Click **Save Settings**

## How It Works

1. **Browse Amazon** — Visit any product page on Amazon
2. **Track** — Click the purple "Track Price" button that appears near the buy box
3. **Monitor** — Price Patrol checks prices in the background at your configured interval
4. **Get Notified** — Receive desktop notifications when prices drop
5. **Earn** — When users click through your notification or popup links, your affiliate tag is attached

## Monetization

Price Patrol uses your Amazon Associates affiliate tag on all outbound links. When users:
- Click a product from the popup
- Click a price drop notification
- Follow any link generated by the extension

Your affiliate tag is attached, earning you **1-10% commission** on qualifying purchases.

## Tech Stack

- **Manifest V3** — Latest Chrome extension standard
- **Chrome Storage API** — Persistent local storage for tracked products
- **Chrome Alarms API** — Reliable background price checking
- **Chrome Notifications API** — Native desktop notifications
- **Vanilla JS** — No frameworks, fast and lightweight

## Privacy

- All data is stored locally in your browser
- No external servers or analytics
- Only communicates with Amazon to check prices
- No tracking or data collection
